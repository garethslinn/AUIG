Accessibility Statement

We are committed to ensuring that our website is fully accessible to all users, regardless of ability or technology. Our goal is to provide an inclusive and user-friendly experience that meets or exceeds current accessibility standards, including the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA.

Accessibility Features

  • Keyboard Navigation

    All interactive elements are fully accessible using the keyboard, with clear focus indicators for improved usability.

  • Screen Reader Support

    Content is structured using semantic HTML, ensuring compatibility with screen readers and assistive technologies. ARIA landmarks and labels are provided for improved navigation.

  • Flexible Text Sizing

    Users can adjust text size using built-in controls (A+ and A-) to suit their preferences.

  • High Contrast Modes

    Multiple themes are available, including:

    • Light Mode
    • Dark Mode
    • Black and White (Minimum Reading) Mode
    These modes enhance readability for users with visual impairments.

  • Responsive Design

    The site is fully responsive and functions across a range of devices, screen sizes, and orientations.

  • Color Accessibility

    Color schemes have been carefully chosen to ensure sufficient contrast. Where edge cases occur, fallback options are provided.

  • Keyboard Focus Management

    Intuitive focus management ensures smooth navigation between sections, improving the experience for keyboard and screen reader users.

  • Skip Links

    A “Skip to Content” link is provided to allow users to bypass repetitive navigation elements.

  • Accessible Forms

    Forms include clear labels, error messages, and ARIA attributes for usability.

  • Customisable Settings

    Users can customise their experience with theme toggles and font size adjustments, ensuring comfort and readability.

  • Content Simplification

    Plain language is used throughout the site to make content easy to understand for a diverse audience.

  • Inclusive Media

    Images include descriptive alt text, and multimedia content provides captions or transcripts where necessary.
